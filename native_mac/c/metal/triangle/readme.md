# Hello, World! in C + Metal

A "Hello, World!" triangle sample using plain C with Apple Metal via an Objective-C++ wrapper.

## Requirements

- macOS with Metal support
- Xcode Command Line Tools (`xcode-select --install`)

## Build

```bash
chmod +x build.sh
./build.sh
```

## Run

```bash
./hello
```
