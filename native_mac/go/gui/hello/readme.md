# Hello, World! in Go + Cocoa GUI

This sample demonstrates a simple "Hello, World!" GUI application in Go using the Cocoa framework via CGo and an Objective-C bridge.

## Requirements

- Go
  ```
  brew install go
  ```

## Build

```bash
sh build.sh
```

This will create `Hello.app` - a macOS application bundle.

## Run

```bash
open Hello.app
```
