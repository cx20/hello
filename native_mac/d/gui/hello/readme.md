# Hello, World! in D + Cocoa GUI

This sample demonstrates a simple "Hello, World!" GUI application in D using the Cocoa framework via an Objective-C bridge.

## Requirements

- D compiler: `dmd`, `ldc2`, or `gdc`
  ```
  brew install dmd
  ```

## Build

```bash
sh build.sh
```

This will create `Hello.app` - a macOS application bundle.

## Run

```bash
open Hello.app
```
