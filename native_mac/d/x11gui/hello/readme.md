# Hello, World! in D + X11 GUI

This sample demonstrates a simple "Hello, World!" GUI application in D using X11 (XQuartz on macOS).

## Requirements

- XQuartz: https://www.xquartz.org/
- D compiler: `dmd`, `ldc2`, or `gdc`
  ```
  brew install dmd
  ```

## Build

```bash
sh build.sh
```

## Run

```bash
sh run.sh
```
