# Hello, World! in D + Vulkan 1.4

Minimal Vulkan 1.4 triangle sample using GLFW and MoltenVK on macOS.

## Requirements
- Homebrew packages: `molten-vk`, `vulkan-loader`, `vulkan-headers`, `glfw`, `glslang`
  ```
  brew install molten-vk vulkan-loader vulkan-headers glfw glslang
  ```
- D compiler (one of): `brew install dmd` or `brew install ldc`

## Build
```bash
sh build.sh
```

## Run
```bash
sh run.sh
```

## Result
(A colored RGB triangle in an 800x600 window)
