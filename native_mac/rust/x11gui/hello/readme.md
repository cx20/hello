# Hello, World! in Rust + X11 GUI

This sample demonstrates a simple "Hello, World!" GUI application in Rust using X11 (XQuartz on macOS) via the `x11` crate.

## Requirements

- XQuartz: https://www.xquartz.org/
- Rust/Cargo
  ```
  brew install rust
  ```

## Build

```bash
sh build.sh
```

## Run

```bash
sh run.sh
```
