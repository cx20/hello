# Hello, World! in Rust + Cocoa GUI

This sample demonstrates a simple "Hello, World!" GUI application in Rust using the Cocoa framework via an Objective-C bridge and FFI.

## Requirements

- Rust compiler
  ```
  brew install rust
  ```

## Build

```bash
sh build.sh
```

This will create `Hello.app` - a macOS application bundle.

## Run

```bash
open Hello.app
```
