# Hello, World! in Rust + Vulkan 1.4

Minimal Vulkan 1.4 triangle sample using ash + winit + MoltenVK on macOS.

## Requirements
- Homebrew packages: `molten-vk`, `vulkan-loader`, `vulkan-headers`, `glslang`
  ```
  brew install molten-vk vulkan-loader vulkan-headers glslang
  ```
- Rust: https://rustup.rs

## Build
```bash
sh build.sh
```

## Run
```bash
sh run.sh
```
