This example demonstrates how to COM programming.

Install:
```
C:\> python -m pip install pywin32
```
Run:
```
C:\> python hello.py
```
Result:
```
+----------------------------------------+
|Browse For Folder                    [X]|

| Hello, COM(Python) Wolrd!              |

| |     :                              | |
| |     :                              | |

| [Make New Folder]    [  OK  ] [Cancel] |
```
