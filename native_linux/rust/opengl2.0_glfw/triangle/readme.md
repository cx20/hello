# OpenGL 2.0 (GLFW) Triangle sample for Linux (Rust)

## Description

OpenGL 2.0 triangle using VBO and GLSL shaders with GLFW window management.

## Requirements

- Rust (rustc, cargo)
- libglfw3-dev

## Build

```sh
$ cargo build --release
```

## Run

```sh
$ ./target/release/hello
```
